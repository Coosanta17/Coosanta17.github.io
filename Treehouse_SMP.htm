The Treehouse SMP Season 3 is a cross-platform Vanilla+ server with land claims. After several months of inactivity, it is back and better than ever! The server has a small friendly community, there is no hacking and cheating, and you can do whatever you want (as long as you follow the rules). This server is about having fun, not who paid the most money for the best rank. Join today!
Join the discord server here.

This server has several extra features that will enhance your experience on the server: Graves (so you wont lose all your items on death), TPA (no more long hours spent visiting your friend), Land claims (don't stress over your base security), Anti-cheat and a few more QOL addons.

Rules:

1.       No Harassment
It's not nice and will make everyone feel bad, including but not limited to: Arguments, Spawn killing, Racism, sexism, homophobia and false denigration.

2.      No cheating
It ruins the fun for everyone.

3.      If a base is claimed, don't destroy it
This includes exploiting the claims system to grief a base. In the wilderness (unclaimed land) you can do whatever you like (except breaking the rules)

4.      Follow the Discord and Minecraft EULA
We don't want any illegals on this server


If you enjoy this server feel free to help keep the server running our Kofi!

A few things to note:


- If your base was griefed, and it wasn't claimed, don't expect any sanctions to be given


- If you feel you have been wrongly punished please don't hesitate to contact one of the Admins or Owners on the discord server and put forward your case


- Currently there is a small world border, so people don't spread out too much, as the server progresses (or the majority votes) the world border will increase.


- There is an issue where some people will time out randomly without any lag or bad ping, we are currently investigating this.


- Most importantly: HAVE FUN!
